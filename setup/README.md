# Setup Restyle

Install the restyle CLI.

## Usage

```yaml
- uses: restyled-io/actions/setup@v3
  with:
    tag:
    # Release tag to install, if empty latest is used
    #
    # Required: false
```

## Inputs

| name  | description                                            | required | default |
| ----- | ------------------------------------------------------ | -------- | ------- |
| `tag` | <p>Release tag to install, if empty latest is used</p> | `false`  |         |

## Outputs

None.
